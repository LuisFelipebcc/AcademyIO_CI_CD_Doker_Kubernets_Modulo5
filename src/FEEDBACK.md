# FEEDBACK

Este arquivo será usado pelo instrutor para consolidar feedbacks. Para adicionar um comentário, abra um Pull Request e edite este arquivo ou adicione um comentário no PR.

Histórico de Interações:

- 21/11/2025: Arquivos iniciais adicionados — `Dockerfile`, `docker-compose.yml`, `payments-api-manifest.yml` (probes e secrets de exemplo), `README.md` e workflow básico de CI.

Recomendações do aluno / instruções ao avaliador:

- Para validar localmente: rode `docker compose up --build` no diretório `src`.
- Se desejar ver o cluster Kubernetes: aplicar o manifest `services/AcademyIO.Payments.API/payments-api-manifest.yml`.
